# Dashboard em Python

Esse é um dashboard feito para a torcida autistas botafoguenses, o url de planilha que está aqui é diferente do url deles para preservação dos dados dos usuários da torcida. No caso o formulário que está sendo analisado nesse sistema é um que eu mesmo fiz copiando o modelo do formulário da torcida.

Link em questão caso queira adicionar algum dado para teste: https://docs.google.com/forms/d/e/1FAIpQLScR2ylHxM-2px4UfDOd3cEnIGo-N3Pfj6miRg71g1FoACINmA/viewform?usp=dialog

Para rodar o script basta baixar a biblioteca do pandas e rodar normalmente como qualquer outro script em Python.

OBS: Qualquer um pode acessar a planilha e ver os dados disponíveis nela através dos códigos desse sistema, portanto recomendo FORTEMENTE que não coloque nenhuma informação pessoal e use apenas informações ficticias para fins de teste.

Obrigado pela atenção! =)
